Configuration

Preferences

Currently there are three preference pages: "Matlab Editor", "Export" and "Helpbrowser".

Most of the options are self-explanatory, so only some of them are described in detail.

Export

This page shares some options with the export dialog. Each confirmed change in this dialog is stored and reappeares in the preference page.

Metadata This group is devided into two sections: general data and filespecific data. Since filespecific data is expacted to change between exports and general data not, only the second part is contained also in the export dialog.
Output Formats
/ export LaTeX core only		 If checked, the exported file will contain only the pure content in LaTeX syntax. This is everything that normally would be placed between \begin{document} and \end{document}. For example, this may be used if one exported Matlab file represents only a single chapter of a bigger LaTeX file.
It's not possible to generate a PDF from a core text.
Transformation options HTML and LaTeX output is generated by transforming an XML file via XSLT files. Here one can specify which transformation files shall be used.

Helpbrowser

Here one can define how a help browser window should appear when using Meditor's Matlab help. Further the basis URL to the Matlab help pages is specified.

Configuration files

Some of Meditor's functionalities are configured by files, which are located in plugindir/config/.
help_funcs This is a simple list of Matlab functions (like matlab_funcs) that have their own help page. Furthermore the helppage's name has to equal the function's name.
helpfileexceptions.xml This is a table of special Matlab expressions (e.g. operators), which links from an expression to its help page.
latexpic.template To include equations in HTML exports, each equation is transformed into a graphic by using LaTeX. This file is a template for this purpose.
matlab_funcs As a simple list of strings this file defines which expressions are Matlab functions. This file may be generated automatically by a tool developed at ITPCP.
toolbox_funcs This file is similar to matlab_funcs but lists toolbox functions.
*.xsl The XSLT files to use for generating HTML and LaTeX output.
