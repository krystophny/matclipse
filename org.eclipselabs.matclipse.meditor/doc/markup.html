The markup langugage

The export functions don't provide their full potential until markup text is used. Markup text is included as matlab comment, but inside the comment a special syntax (based on Mediawiki) is used to declare a markup text. Meditor (and all other editors) displays such sections as standard matlab comment, but in exported files markup text represents headings, links, bold text, ...

Matlab code is displayed in highlighted boxes. Each markup content breaks into such a box and occures like ordinary text. So it's also possible to write a document that consists mostly of normal text and some matlab code.

This is a list of all implemented markup definitions.

Element Syntax
Comments
Headings %==heading1==
%===heading2===
%====heading3====
Markup text %% text
Creates standard text in proportional typeface.
Truetype text %% text <tt>variable</tt> and text again
Link %% text [http://something.somedomain] and more text
Equation %% $$ this = formula $$
The equation will be printed centered in its own line without a serial number.
Unnumbered list %* first entry
%* next entry
Numbered list %# first entry
%# next entry
It's not possible to interleave lists (ordered and unordered).
LaTeX command %\command
These commands are only interpreted when exporting to LaTeX or PDF. It's possible to use every LaTeX command, so the author is responsible for seeing that they are applied in a reasonable way. One has also to keep in mind that the whole line will be seen as LaTeX command.
